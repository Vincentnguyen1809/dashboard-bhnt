Usage Instructions

1. Basic Usage

// Create a new ActivityLog instance
const activityLog = new ActivityLog({
  containerId: 'activity-log-container',
  activities: myActivitiesArray,
  itemsPerPage: 10  // Optional, defaults to 10
});

2. Data Format

const activities = [
  {
    id: 'ACT-001',
    content: 'Task completed',
    user: 'John Doe',
    timestamp: new Date(),
    status: 'completed'  // completed, edited, deleted, created, updated, pending
  },
  // ... more activities
];

3. Update Activities

// Update with new data
activityLog.updateActivities(newActivitiesArray);

4. Integration with Your Existing Code

// In your index.html, add the script tag:
<script src="ActivityLog.js"></script>

// Add container element:
<div id="activity-log-container"></div>

// Initialize in your main script:
const activityLog = new ActivityLog({
  containerId: 'activity-log-container',
  activities: activityLogs,  // Your existing activityLogs array
  itemsPerPage: 10
});

// Update when data changes:
// (e.g., in your Firestore onSnapshot callback)
onSnapshot(collection, (snapshot) => {
  const activities = snapshot.docs.map(doc => doc.data());
  activityLog.updateActivities(activities);
});
